# Maintenance Backend

Backend service for industrial maintenance management.

## 📌 Description

This service manages production lines and maintenance records, including:

- Lines
- Machines
- Maintenance plans
- Maintenance scheduling
- Due dates and snooze logic

Built with:

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

---

## ⚙️ Requirements

- Java 21
- PostgreSQL 18+
- Maven (or IntelliJ with Maven support)
